Pour compiler le programme tapez "make".
Pour éxecuter le programme tapez "./executable".
Un fichier de donnée est requis pour l'éxecution du programme, celui-ci dois se trouvé dans le même dossier que le programmme.
Le format du fichier de donnée doit être : nom x1  x2  x3 ...(pour chaque ligne) ou  avec des labels en plus en haut que chaque colonne(numérique).
Après l'éxecution du programme utilisez gnuplot<gnu.gnu pour lancé le script et le rendu se trouve dans le fichier graph.png
